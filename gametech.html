Program Game Technology

Game Technology adalah program studi yang berfokus pada pengembangan teknologi interaktif, mencakup desain game, pemrograman, animasi, dan implementasi teknologi modern seperti AR, VR, dan simulasi digital. Program ini dirancang untuk melatih mahasiswa memahami bagaimana sebuah game dibangun mulai dari konsep, alur cerita, gameplay, hingga implementasi teknis dan pengujian performa.

Mahasiswa akan dibekali kemampuan membuat game menggunakan berbagai game engine seperti Unity dan Unreal Engine. Selain itu, mahasiswa juga mempelajari grafika komputer, artificial intelligence untuk game, desain karakter, audio game, hingga optimalisasi performa. Kombinasi ini membuat lulusan mampu menghasilkan game yang tidak hanya menarik secara visual, tetapi juga stabil, interaktif, dan inovatif.

Dengan kebutuhan industri game yang terus meningkat, program ini membekali mahasiswa agar siap bersaing di dunia profesional, baik sebagai game developer, game designer, 3D artist, animator, maupun AR/VR developer. Lulusan diharapkan mampu menciptakan produk game yang kreatif, berkualitas tinggi, dan relevan dengan perkembangan industri hiburan digital global.
